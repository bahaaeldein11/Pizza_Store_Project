<h1>Pizza_Store_Project</h1>
<h4>Problem Statment:</h4>
This project focuses on analyzing pizza sales data using Power BI to identify key performance indicators, evaluate business performance, and uncover trends related to revenue, order volume, and customer behavior.
The following metrics will be calcualted to get insghts from the data which contains record from Jam/2015 to Dec/2015.
<ul>
  <li>Total Revenue</li>
  <li>Average Order Value</li>
  <li>Total Pizzas Sold</li>
  <li>Total Orders</li>
  <li>Average Pizzas Per Order</li>
</ul>
